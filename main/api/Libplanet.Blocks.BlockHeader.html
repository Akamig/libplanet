public BlockHeader(int protocolVersion, long index, DateTimeOffset timestamp, Nonce nonce, Address miner, long difficulty, BigInteger totalDifficulty, BlockHash? previousHash, HashDigest<SHA256>? txHash, BlockHash hash, ImmutableArray<byte> preEvaluationHash, HashDigest<SHA256>? stateRootHash)
Parameters
Type Name Description
Int32 protocolVersion

The protocol version. Goes to ProtocolVersion.
Int64 index

The height of the block. Goes to Index.
DateTimeOffset timestamp

The time the block is created. Goes to Timestamp.
Nonce nonce

The nonce which satisfies given difficulty. Goes to Nonce.
Address miner

The address of the miner. Goes to Miner.
Int64 difficulty

The mining difficulty that nonce has to satisfy. Goes to Difficulty.
BigInteger totalDifficulty

The total mining difficulty since the genesis, including the block's difficulty. See also Difficulty.
Nullable<BlockHash> previousHash

The previous block's Hash. If it's a genesis block (i.e., its Index is 0) this should be null. Goes to PreviousHash.
Nullable<HashDigest<SHA256>> txHash

The result of hashing the transactions the block has. Goes to TxHash.
BlockHash hash

The hash digest derived from the whole contents of the block including stateRootHash, which is determined by evaluating transactions. This is used for block's unique identifier. Goes to Hash.
ImmutableArray<Byte> preEvaluationHash

The hash derived from the block excluding stateRootHash (i.e., without action evaluation). Used for checking nonce. See also Libplanet.Blocks.BlockHeader.Validate(Libplanet.HashAlgorithmType,System.DateTimeOffset).
Nullable<HashDigest<SHA256>> stateRootHash

The Hash of the resulting states after evaluating transactions and a block action (if exists).
Remarks

This is only exposed for testing. Should not be used as an entry point to create a BlockHeader instance under normal circumstances.
