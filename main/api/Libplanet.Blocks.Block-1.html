Block(Int64, Int64, BigInteger, Nonce, Nullable<Address>, Nullable<BlockHash>, DateTimeOffset, IReadOnlyList<Transaction<T>>, Nullable<BlockHash>, Nullable<HashDigest<SHA256>>, Int32)

Creates a Block<T> instance by manually filling all field values. For a more automated way, see also Mine(Int64, Int64, BigInteger, Address, Nullable<BlockHash>, DateTimeOffset, IEnumerable<Transaction<T>>, Int32, CancellationToken) method.

Declaration
public Block(long index, long difficulty, BigInteger totalDifficulty, Nonce nonce, Address? miner, BlockHash? previousHash, DateTimeOffset timestamp, IReadOnlyList<Transaction<T>> transactions, BlockHash? preEvaluationHash = default(BlockHash? ), HashDigest<SHA256>? stateRootHash = default(HashDigest<SHA256>? ), int protocolVersion = 1)
Parameters
Type Name Description
Int64 index

The height of the block to create. Goes to the Index.
Int64 difficulty

The mining difficulty that nonce has to satisfy. Goes to the Difficulty.
BigInteger totalDifficulty

The total mining difficulty until this block. See also Difficulty.
Nonce nonce

The nonce which satisfy the given difficulty with any other field values. Goes to the Nonce.
Nullable<Address> miner

An optional address refers to who mines this block. Goes to the Miner.
Nullable<BlockHash> previousHash

The previous block's Hash. If it's a genesis block (i.e., index is 0) this should be null. Goes to the PreviousHash.
DateTimeOffset timestamp

The time this block is created. Goes to the Timestamp.
IReadOnlyList<Transaction<T>> transactions

The transactions to be mined together with this block. Transactions become sorted in an unpredicted-before-mined order and then go to the Transactions property.
Nullable<BlockHash> preEvaluationHash

The hash derived from the block except of stateRootHash (i.e., without action evaluation). Automatically determined if null is passed (which is default).
Nullable<HashDigest<SHA256>> stateRootHash

The Hash of the states on the block.
Int32 protocolVersion

The protocol version. CurrentProtocolVersion by default.
